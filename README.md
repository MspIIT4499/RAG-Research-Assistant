# ResearchHub AI

An AI-powered Research Paper Assistant using Retrieval-Augmented Generation (RAG), semantic search, vector embeddings, and Gemini.

## Features

- Upload multiple research papers
- Semantic search
- Citation-aware answers
- Multi-document support
- Paper summarization
- Paper comparison

## Tech Stack

- Python
- Streamlit
- LangChain
- Sentence Transformers
- FAISS
- Gemini API
- PyMuPDF
